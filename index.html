How to Use

Audio — Click ▶ to hear pronunciation via system text-to-speech. If no Hebrew voice is installed (Microsoft Asaf on Windows, Carmit on Mac), a banner appears at the top with install instructions, and ▶ buttons become Forvo links (real native recordings on forvo.com).
Niqqud Toggle — Lessons 03 onwards have a Niqqud ON / OFF toggle in the header. Default is ON for learning correct pronunciation. Switch OFF to read like real Israelis (no vowel marks). What ulpans actually teach: niqqud for the first 3 weeks, then progressively without.
Reading direction — Hebrew is read right-to-left. Hebrew text in cards is right-aligned and flows from right to left within each word.
Progress — Lesson completion and quiz scores stored in your browser's localStorage. No account, no tracking, no server.
