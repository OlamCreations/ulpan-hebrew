Object Pronouns

Lesson 24 · כינויי מושא
Home24

When a verb has a definite direct object ("I see the book"), Hebrew uses את (et) before it. When the object is a pronoun ("I see him"), the pronoun fuses with את: oti, otcha, oto, ota, otanu, etchem, otam.

The marker את (et) — Used only before definite direct objects (with ה, proper names, or pronouns). אֲנִי רוֹאֶה אֶת הַסֵּפֶר (ani ro'e et ha-sefer) "I see the book". But: אֲנִי רוֹאֶה סֵפֶר (no et, indefinite).

Possessive Suffixes (alternative to "shel")

Two ways to say "my" — Modern: הַסֵּפֶר שֶׁלִּי (ha-sefer sheli) "my book". Literary/biblical: סִפְרִי (sifri) — same meaning, fused suffix. Used in poetry, prayers, set phrases.
